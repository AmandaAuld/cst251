Challenge

The challenge of this lab was to create a responsive layout using the flexbox layout method as well as a Media Query Rule.

Problems

We had to experiment with the layout and position of the various elements on the webpage. We referred to the lecture when adding the Media Query Rule in order to make sure the page functioned properly.

Result

We successfully created a responsive webpage using the flexbox layout method and a Media Query Rule that shifts the elements to fit devices with smaller screens.
