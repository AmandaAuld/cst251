Challenge

The purpose of this lab was to create a layout for our lab page using CSS layout tools.

Problems

It took time to understand how the different CSS layouts worked and functioned. We had to experiment, revise, and refer back to the lecture multiple times in order to complete the lab. CSS float proved to be especially difficult and less responsive than flexbox and grid.

Result

We were able to style different versions of the same webpage using CSS float, CSS flexbox, and CSS grid.
